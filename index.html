  1. Goal #1: Our own websites on Swarm

    In one sentence, the goal to achieve
    We upload and can download our own websites (swarm.eth, docs.swarm.eth), through every Swarm product (bee, gateway, cli, chrome extension, dashboard/desktop).

    1. Measuring success

      • 5 MB Zipped file w html, css, js.
      • Can be uploaded/downloaded through the gateway, swarm-cli, dashboard/desktop, bee-curl
      • 70% of 5MB website uploads take up-to 2 minutes
      • 70% of 5mb website downloads take up-to 20 seconds
      • The websites can be accessed through a human name w/o doing a blockchain tx for every update

      ETA: 21 Oct 2021 +3 months

    2. Value

      How does the world benefit, when this goal is achieved:
      Publishers and other organisations have a way to publish information in a non-censorable way. This guarantees freedom of information, which is a human right. If we meet this goal, data can start flowing through the network which in itself provides us with the information needed to test and improve the different software tools.

  2. Goal #2: It’s easy and understandable to set up and interact with a Bee node

    Any crypto enthusiast with an interest in Swarm can set up and run a Bee node easily.

    1. Measuring Success

      • Following the documentation or following a user journey, setting up a node on Mac, Windows and Linux can be done in under 30’ in >70% of the cases.
      • This will be measured by witnessing newly-joined Discord members set up their node.
      • Onboarding to tokens etc via docs
      • Reasonable security for api’s ???
      • It works on Mainnet and testnet
      • Insight of the workings of the Bee node
      • Control over basic parameters

    2. Timebox

      Goal#1 + 2 months

    3. Value

      how does the world benefit, when this goal is achieved
      If it is trivial to set up a Bee node, more of them will be set up. This will result in more node operators, and a stronger network.

  3. Goal #3: Keep my data on Swarm for as long as I choose (incentives)

    In one sentence, the goal to achieve
    I can pay to have my content persisted on Swarm.
    This payment is distributed to node operators

    1. Measuring success

      • Upload a 5MB file like a website (Goal #1),
      • Pay for a stamp in x interface
      • Check every day for 2 weeks if it is still there? :) And it’s still there
      • If the network is not under attack
      • I have visibility on the status / value of my postage stamp / incentive value
      • If the lifespan of your node is long enough you will “profit” from the revenue redistribution of wealth in Swarm network
      • As a node operator I can monitor my earnings

    2. Nice to have

      • No staking, no challenge, no commit reveal

    3. Value

      how does the world benefit, when this goal is achieved
      Archives, backups, …
